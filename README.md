# Eduardo Silva | Portfólio de Designer Gráfico

Landing Page moderna, focada em conversão para o designer Eduardo Silva.

## Tecnologias
- React 19 + Vite
- TypeScript
- Tailwind CSS (Estilização Premium)
- Framer Motion (Animações Sutis)
- Lucide React (Ícones)
- React Helmet Async (SEO Dinâmico)

## Configuração
Renomeie `.env.example` para `.env` e preencha as variáveis de contato:
- `VITE_WHATSAPP_NUMBER`: Número no formato internacional (ex: 5511999999999)
- `VITE_INSTAGRAM`: Link do perfil
- `VITE_LINKEDIN`: Link do perfil
- `VITE_EMAIL`: E-mail de contato

## Como Rodar
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
3. Construa para produção: `npm run build`

## Diferenciais Técnicos
- Sistema de temas (Light/Dark) persistente.
- SEO completo com meta tags e JSON-LD Schema.
- Performance otimizada com lazy loading e renderização estratégica.
- Mobile-first e acessibilidade ARIA.
